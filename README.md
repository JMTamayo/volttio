# Volttio

Watt’s happening? Now you’ll know for sure! Unmask the energy ghosts in your home with Volttio.

## Firmware

- **alpha** — Lab firmware on ESP32 (PlatformIO): BLE provisioning, Wi‑Fi, MQTT, PZEM energy metrics. See `alpha/README.md`.
- **lumen** — ESP32‑C6 firmware with ESP‑IDF via PlatformIO (RGB / peripheral experiments). See `lumen/platformio.ini` and sources under `lumen/src/`.
