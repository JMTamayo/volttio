# Volttio

Watt’s happening? Now you’ll know for sure! Unmask the energy ghosts in your home with Volttio.

## Firmware

- **alpha-lab** — Lab firmware on ESP32 (PlatformIO): BLE provisioning, Wi‑Fi, MQTT, PZEM energy metrics. See `alpha-lab/README.md`.
- **glow-lab** — ESP32‑C6 firmware with ESP‑IDF via PlatformIO (RGB / peripheral experiments). See `glow-lab/platformio.ini` and sources under `glow-lab/src/`.
